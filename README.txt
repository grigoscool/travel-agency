Тренировочный сайт для турагенства.

Реализованный функционал:

- карусель с картинками и описанием мест, продаваемых туроператором
- информационные карты и таблички с ценами и видами услуг
- форма для обратной связи, с подключенным телеграм ботом для отправления сообщений о заказе
- настройка админ панели
